### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hkuspace-pu/COMP-3000-Ngan-Tsz-Chung-FYP
   ```
2. Install NPM packages
   ```sh
   yarn preinstall
   ```
3. Run application
   ```sh
   yarn start
   ```
